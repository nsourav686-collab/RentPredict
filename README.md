# 🏠 RentPredict – AI-Based Room Rent Prediction System 📌 Overview

RentPredict is an AI-powered web application that predicts house rent based on property details and location using Machine Learning.

✨ Features

- 🔐 User Registration & Login
- 🔑 Google OAuth Login
- 🏡 Property Management
- 🤖 AI-Based Rent Prediction
- 📍 Google Maps Integration
- 📊 Market Insights
- 🔔 Rent Alerts
- 👤 Owner Dashboard
- 🛠️ Admin Dashboard
- 📱 Responsive Design

 🛠️ Technologies Used

 Frontend
- React.js
- Vite
- HTML5
- CSS3
- Tailwind CSS
- JavaScript

### Backend
- Python
- Flask

### Database
- SQLite

### Machine Learning
- Scikit-learn
- Pandas
- NumPy

## 🚀 Installation

```bash
git clone https://github.com/nsourav686-collab/RentPredict.git
```

```bash
cd RentPredict
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run

```bash
python MAIN/app.py
```
